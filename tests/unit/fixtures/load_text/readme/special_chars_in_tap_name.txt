# Homebrew Luciferous Tap

Personal Homebrew tap for my tools

## Installation

```bash
brew tap user-name/homebrew-special-tap-name
```

## Available Casks

### my-special-tool

A special tool

```bash
brew install --cask my-special-tool
```

## Updating

```bash
brew update
brew upgrade <formula>
```

## Uninstalling

```bash
brew uninstall <formula>
brew untap user-name/homebrew-special-tap-name
```
